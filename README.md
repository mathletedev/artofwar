# Art of War

C++ program to display Sun Tzu quotes!

## Usage

Clone the repo to your local machine:

```bash
git clone https://github.com/mathletedev/artofwar
```

Build the source code:

```bash
g++ -o artofwar artofwar.cpp
```

Run the program!

```bash
./artofwar
```
